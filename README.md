# Francis Waweru Mwaura Portfolio

## Overview
This is the portfolio website for Francis Waweru Mwaura, a full-stack developer based in Kenya. The website showcases his skills, projects, and provides a way for potential clients or collaborators to get in touch.

## Project Structure
The project consists of the following files and directories:

```
francis-portfolio
├── assets
│   └── profile-placeholder.svg
├── css
│   └── styles.css
├── js
│   └── main.js
├── index.html
├── projects.html
├── contact.html
└── README.md
```

### File Descriptions
- **index.html**: The homepage of the portfolio. It includes sections for the hero, about me, skills, projects, testimonials, and contact information.
  
- **projects.html**: A dedicated page that showcases the projects completed by Francis Waweru Mwaura. It includes a list of projects with descriptions and links to live demos or GitHub repositories.

- **contact.html**: This page contains a contact form that allows visitors to send messages to Francis. It includes fields for the user's name, email, and message.

- **assets/profile-placeholder.svg**: A placeholder image for the profile photo used in the homepage.

- **css/styles.css**: This file contains the styles for the entire portfolio website, including layout, typography, and responsive design.

- **js/main.js**: This file includes JavaScript functionality for the website, such as smooth scrolling, form validation, and dark/light mode toggle.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the homepage.
3. Navigate to the `projects.html` and `contact.html` pages to explore the projects and contact form.
